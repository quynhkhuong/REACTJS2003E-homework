#Tổng_hợp buổi hôm nay:
git clone (chỉ dùng download project lần đầu)
git add [tên folder/tên file] (thêm files vào danh sách sẽ chuẩn bị được đưa lên server)
git commit -m "Nội dung của thay đổi" (tạo ghi chú cho các thay đổi của mình)
git push origin master (upload code lên server, upload vào nhánh master)
git pull origin // lấy code trên server về
git status // kiểm tra trạng thái code hiện tại (đang ở branch nào, có những thay đổi gì, ...)

Ngoài ra còn có 1 số lệnh như:
git branch [branch_name] // tạo 1 nhánh mới
git checkout [branch_name] // chuyển sang 1 nhánh khác
git merge branch_A // lấy các thay đổi của nhánh branch_A áp dụng vào nhánh hiện tại
